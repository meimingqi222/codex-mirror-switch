# 项目文档

本目录包含 codex-mirror-switch 项目的相关文档。

## 文档列表

### [code-review-bugs.md](./code-review-bugs.md)
代码审阅报告 - 记录了项目中发现的所有逻辑bug和问题，包括：

- 🔴 **严重问题**: 竞态条件、状态不一致等需要立即修复的问题
- 🟡 **中等问题**: 环境变量清理、软删除恢复等功能性缺陷
- 🟢 **轻微问题**: 错误处理、代码复杂度等可优化的地方

每个问题都包含：
- 问题描述和影响分析
- 相关代码片段和文件位置
- 详细的修复建议
- 优先级标记

### 文档说明

- 所有文档都使用 Markdown 格式
- 包含详细的代码示例和分析
- 提供可操作的修复建议
- 按照严重程度和优先级分类

### 贡献指南

在修复这些问题时，请：

1. 参考 `code-review-bugs.md` 中的详细分析
2. 按照优先级顺序处理问题
3. 添加相应的单元测试
4. 更新文档记录修复状态
5. 进行回归测试确保不引入新问题
